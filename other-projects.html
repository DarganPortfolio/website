Graph Reachability Visualization Interface



Description

The Graph Reachability Visualization Interface is an interactive tool designed to help users understand graph theory and search algorithms. Users can create and visualize graphs, using algorithms like Depth-First Search (DFS) and Breadth-First Search (BFS) to find and highlight paths between nodes. The app dynamically shows the traversal process, marking intermediate nodes and edges to illustrate how the algorithm works. With an intuitive UI built using modern web technologies and visualization libraries, this app serves as an educational tool for learning graph theory, analyzing algorithms, and exploring complex graph data.


Skills Used

  • Java
  • User Interface (UI) Design
  • Algorithm Implementation
  • Data Structures
  • Data Visualization

Dynamic Terrain Height Map Generator Application



Description

The World Height Map Generator is an application that employs a Perlin noise algorithm to generate detailed greyscale images representing height maps. Users have the flexibility to scale and adjust the layout, ensuring precise control over the terrain's topography. These height maps can be saved and seamlessly imported into various applications, including photoshoots, 3D modeling softwares, and landscape generation, providing a versatile tool for artists, designers, and developers. The app's intuitive interface and functionality makes it ideal for creating realistic and customizable landscapes, enhancing creative projects with high-quality terrain data.


Skills Used

  • Java
  • User Interface (UI) Design
  • Perlin Noise Algorithms
  • Image Processing
  • Data Export
